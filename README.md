# PRIS-2026: Анализ расходов по чекам

## Описание проекта

**Название:** Система анализа личных расходов на основе фотографий чеков  
**Тема:** Автоматизация учёта личных финансов с помощью компьютерного зрения и обработки естественного языка

**Проблема, которую решает проект:**  
Большинство людей тратят время на ручной ввод данных из чеков в приложения или таблицы. Наша система позволяет просто сфотографировать чек → текст автоматически извлекается → сумма и товары распознаются → расходы категоризируются и визуализируются в удобном веб-интерфейсе.

**Цель:** Создать прототип веб-приложения, которое помогает пользователю вести учёт трат без ручного ввода данных.

## Технологический стек (Tech Stack)

- **Интерфейс пользователя** — Streamlit  
- **Обработка изображений и OCR** — OpenCV + (в будущем Tesseract / EasyOCR / облачный OCR)  
- **Обработка текста и категоризация** — pandas, (в будущем — NLP-модели или простые правила)  
- **Хранение данных** — pandas DataFrame / (в будущем — SQLite)  
- **Язык программирования** — Python 3.9+  
- **Дополнительно** — numpy, matplotlib / plotly для графиков
